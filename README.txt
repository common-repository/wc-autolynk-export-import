This is a plugin for Wordpress and WooCommerce that will automate the production of shipping labels within AnPost’s AutoLynk software. The plugin will also automate the sending of tracking numbers to customers after the AutoLynk software has generated them.

== Description ==

The VGP Import Export for Autolynk in WooCommerce plug-in bridges the gap between An Post’s Autolynk software and WooCommerce. Using this plug-in you can vastly speed up the shipping of customer orders and avoid manual data entry for tracking numbers. All this while taking advantage of An Post’s great national and international postage rates for smaller packages.

= What is Autolynk? =

Autolynk is a software package provided by An Post, the national Post Office of the Republic of Ireland, to small and medium businesses. The software prints pre-paid shipping labels for your national and international customers. To get Autolynk, please contact your An Post account manager.

== Installation ==

= Automatic Installation =
Go to Plugins > Add New from your WP admin menu, search for Woo WC Autolynk Export Import
, install, and activate.

= Manual Installation =
Download the plugin and unzip it.
Upload the plugin folder to your wp-content/plugins folder.
Activate the plugin from the plugins page in the admin.
